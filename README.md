Repositorio personal con la resolución de guías de ejercicios y material de estudio de la materia **Paradigmas de Programación** (TB025 - Cátedra Essaya) de la carrera Ingeniería Informática (UBA).

---

## 📌 Objetivo

El objetivo de este repositorio es documentar el proceso de aprendizaje de los distintos paradigmas de programación, abordando tanto los conceptos teóricos como su aplicación práctica mediante ejercicios.

Se busca:

- Consolidar los fundamentos de cada paradigma
- Desarrollar código claro, mantenible y bien estructurado
- Comprender las diferencias, ventajas y limitaciones de cada enfoque
- Aplicar principios de diseño en la resolución de problemas

---

## 🧩 Contenidos de la materia

A lo largo de las guías se trabajan los siguientes temas:

### 🔹 Paradigmas fundamentales
- Concepto de paradigma de programación
- Comparación entre enfoques

### 🔹 Programación orientada a objetos
- Objetos, estado y comportamiento
- Encapsulamiento
- Abstracción
- Herencia y polimorfismo
- Interfaces y clases abstractas
- Relaciones entre clases
- UML (diagramas de clases, secuencia, actividades, casos de uso)

### 🔹 Principios de diseño
- Buenas prácticas de modelado
- Separación de responsabilidades

### 🔹 Programación orientada a eventos
- Manejo de eventos
- Interfaces gráficas

### 🔹 Programación lógica
- Resolución de problemas declarativa

### 🔹 Cálculo Lambda
- Fundamentos teóricos de la programación funcional

### 🔹 Programación funcional
- Funciones puras
- Inmutabilidad
- Funciones de orden superior
- Composición de funciones
- Recursión

### 🔹 Programación concurrente
- Conceptos básicos de concurrencia
